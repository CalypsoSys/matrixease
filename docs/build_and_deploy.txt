Build Ubuntu Desktop 
=====================
01. Start "Clone of MatrixEase Manga Ubuntu 64-bit - Development" vm
02. Open a terminal
03. cd ~/dotnet/MatrixEase/
04. git pull
05. cd web_blaster
06. dotnet build --configuration Release
07. cd ../MatrixEase.App
08. sudo rm -r obj/
09. sudo rm -r bin/
10. Make sure a valid appsettings.json file is there!!!!
11. electronize build /target linux
12. cp ~/dotnet/MatrixEase/MatrixEase.App/bin/Desktop/matrixease_1.0.0_amd64.snap ~/.
13. snapcraft login
14. snapcraft upload --release=beta ~/dotnet/MatrixEase/MatrixEase.App/bin/Desktop/matrixease_1.0.0_amd64.snap
15. copy ~/matrixease_1.0.0_amd64.snap to c:\transfer
16. from windows copy "<artifact-staging-path>\matrixease_1.0.0_amd64.snap" "<repo-root>\www.matrixease.com\download\matrixease_1.0.0_amd64.snap"


Build Docker Image
==================
01. cd ~/dotnet/MatrixEase/
02. git pull
03. cd web_blaster
04. dotnet build --configuration Release
05. cd ..
06. ~/dotnet/MatrixEase/bin/web_blaster Ubuntu Release ~/dotnet/MatrixEase/static.matrixease.wwwroot ~/dotnet/MatrixEase/MatrixEase.Web/wwwroot googs
07. sudo docker build -t matrixease_web .
08. sudo docker rmi -f $(sudo docker images -f "dangling=true" -q)
09. sudo docker save matrixease_web > ~/matrixease_web.tar
10. copy ~/matrixease_web.tar to <artifact-staging-path>


Deploy Static Sites
===================
01. Treat www.matrixease.com and docs.matrixease.com as plain static site roots.
02. Publish each directory directly with Cloudflare Pages or equivalent static hosting.
03. No Docker image or container deployment is required for those two sites.
04. Keep links relative inside each site and avoid server-specific config files.


Init Docker Image
=================
01. cd ~/docker
02. ./docker_matrixease.sh stop
03. mv matrixease_web.tar matrixease_web.tar.lastgood
04. copy matrixease_web.tar to <deploy-user-home>/docker (on deployment machine)
05. cd matrixease_dotnet
06. sudo docker-compose down
07. cd ..
08. sudo docker load < matrixease_web.tar
09. sudo docker rmi -f $(sudo docker images -f "dangling=true" -q)
10. cd matrixease_dotnet
11. sudo docker-compose up -d
12. sudo docker-compose stop
13. cd ..
14. ./docker_matrixease.sh start
