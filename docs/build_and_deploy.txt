Build Ubuntu Desktop 
=====================
01. Start "Clone of MatrixEase Manga Ubuntu 64-bit - Development" vm
02. Open a terminal
03. cd ~/dotnet/MatrixEase/
04. git pull
05. cd web_blaster
06. dotnet build --configuration Release
07. cd ../MatrixEase.App
08. sudo rm -r obj/
09. sudo rm -r bin/
10. Make sure a valid appsettings.json file is there!!!!
11. electronize build /target linux
12. cp ~/dotnet/MatrixEase/MatrixEase.App/bin/Desktop/matrixease_1.0.0_amd64.snap ~/.
13. snapcraft login
14. snapcraft upload --release=beta ~/dotnet/MatrixEase/MatrixEase.App/bin/Desktop/matrixease_1.0.0_amd64.snap
15. copy ~/matrixease_1.0.0_amd64.snap to c:\transfer
16. from windows copy "C:\transfer\matrixease_1.0.0_amd64.snap" "C:\CalypsoSystems\MatrixEase\www.matrixease.com\download\matrixease_1.0.0_amd64.snap"


Build Docker Image
==================
01. cd ~/dotnet/MatrixEase/
02. git pull
03. cd web_blaster
04. dotnet build --configuration Release
05. cd ..
06. ~/dotnet/MatrixEase/bin/web_blaster Ubuntu Release ~/dotnet/MatrixEase/static.matrixease.wwwroot ~/dotnet/MatrixEase/MatrixEase.Web/wwwroot googs
07. sudo docker build -t matrixease_web .
08. sudo docker rmi -f $(sudo docker images -f "dangling=true" -q)
11. sudo docker save matrixease_web > ~/matrixease_web.tar
12. copy ~/matrixease_web.tar to c:\transfer
15. from windows copy "C:\transfer\matrixease_web.tar" "C:\CalypsoSystems\MatrixEase\www.matrixease.com\download\matrixease_web.tar"


Init Docker Image
=================
01. cd ~/docker
02. ./docker_matrixease.sh stop
03. Update  matrixease_docs_static/htdocs/ with all files from docs.matrixease.com
04. Update matrixease_static/htdocs/ with all files from www.matrixease.com
05. mv matrixease_web.tar matrixease_web.tar.lastgood
06. copy matrixease_web.tar to /home/calypso/docker (on deployment machine)
07. cd matrixease_dotnet
08. sudo docker-compose down
08. cd ..
09. sudo docker load <  matrixease_web.tar
10. sudo docker rmi -f $(sudo docker images -f "dangling=true" -q)
11. cd matrixease_dotnet
12. sudo docker-compose up -d
13. sudo docker-compose stop
14. cd ..
15. ./docker_matrixease.sh start
