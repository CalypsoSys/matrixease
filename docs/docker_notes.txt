Build Docker Image (from https://github.com/CleaninterNet/filter14.git)
=======================================================================
 1. cd ~/dotnet
 2. git clone git@github.com:CalypsoSys/inctrak_manga.gitpwd
 3. cd ~/dotnet/inctrak_manga/
 4. git pull or git checkout "branch I want"
 5. Build blaster
 6. ~/dotnet/inctrak_manga/bin/web_blaster Ubuntu Release ~/dotnet/inctrak_manga/static.manga.inctrak ~/dotnet/inctrak_manga/visalyzer.web/wwwroot
 7. sudo docker build -t visalyzer.web .
 8. sudo docker rmi -f $(sudo docker images -f "dangling=true" -q)
 9. sudo docker save visalyzer.web > /tmp/visalyzer.web.tar
10. copy visalyzer.web.tar /home/???/docker (on deployment machine)


Init Docker Image
==================
 1. cd ~/
 2. mkdir Docker
 3. cd Docker
 4. mkdir visalyzer_web_dotnet
 5. cd visalyzer_web_dotnet
 6. copy over Docker/docker-compose.yml
 7. mkdir app
 8. copy over appsettings_prod_docker.json to app as appsettings.json
