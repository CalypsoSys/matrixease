Quick Start

Get a dataset into MatrixEase and start analyzing.

Use the hosted workspace for the fastest start, or use the desktop build when you want a local workflow.

1. Choose how to access MatrixEase

  • Hosted workspace for browser access.
  • Windows and Ubuntu downloads for desktop access.
  • Contact the team directly for larger deployment or integration needs.

2. Load a source file or Google Sheet

MatrixEase supports CSV, Excel workbooks, and Google Sheets. For CSV inputs, choose the separator that matches the file. For Google Sheets, provide the sheet identifier and range.

3. Review the generated matrix

After loading, MatrixEase categorizes the incoming data and presents the matrix view for filtering, metric review, and relationship discovery.

4. Register or sign in when needed

The hosted workflow may ask you to sign in with email or Google so saved projects can be associated with your account.
