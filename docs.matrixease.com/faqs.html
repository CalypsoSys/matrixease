FAQs

Common product, support, and deployment questions.

Is MatrixEase free?

There are free usage paths for evaluation and smaller workloads. Contact the team for licensing beyond those limits.

What data sources are supported?

CSV, Excel workbooks, and Google Sheets are the documented starting points.

Can I use MatrixEase without the hosted workspace?

Yes. Desktop builds are available for Windows and Ubuntu.

Can I run it on-premises?

Contact the team for enterprise deployment and integration planning.

How do I request a feature?

Use features@matrixease.com or the support site.

How do I contact support?

Email support@matrixease.com or open a request through the support center.

Do I need to recategorize data manually?

No. MatrixEase is designed to categorize source values automatically and then let you refine the analysis through filtering.

Where can I learn the workflow faster?

Start with the Quick Start page, then use the video walkthroughs for the major flows.
